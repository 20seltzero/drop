How to Use

Drop is an open-source javascript and CSS library which makes amazing dropdowns. It's powered by Tether.js.

To use, first download the latest release.

Then simply add this to your page:

<link rel="stylesheet" href="drop-theme-arrows.css" />
<script src="drop.min.js"></script>

And that's it! Learn more by visiting the documentation.
